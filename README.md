# Snake

A simple implementation of the old video game **Snake**.

<img src="img/screenshot_1000.gif?raw=true" height="300">


## How to

run ```main.py```


## Dependencies

- Python v3.10.9
- Numpy v1.24.1
- PyGame v2.1.2
- PyTorch v1.13.1
- Tqdm v4.64.1
- Matplotlib v3.6.2
- Typer v0.7.0

Install them with ```pip install -r requirements.txt```
